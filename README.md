# Farhan Portfolio

React frontend developer portfolio with a premium hacker-style UI.

## Available Commands

- `npm install`
- `npm start`

## Project Structure

- `public/index.html` — app shell and fonts
- `src/App.js` — portfolio layout and sections
- `src/App.css` — light blue background, navigation, footer, and animation styles
- `src/index.js` — React entry point
